**Tim Berners-Lee** invented WWW and HTML

- Hyper Text Markup Language
- HTML provides structure for a Web Page
- HTML elements tells browser how to display the content
- The elements label pieces of contents as 'headings', 'paragraphs', 'body', 'link' etc.

---

**Basic HTML structure:**

```
<!DOCTYPE html>
<html>
<head>
<title>This is a Title </title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph</p>
</body>
</html>
```

**Important Elements:**
`<!DOCTYPE html>` - declares that the document is a HTML document
`<html>` - root of HTML document
`<head>` - has the meta information about the HTML page
`<body>` - defines body of the page and acts as a container for paragraphs, headers, tables, images etc.
`<h1>`- headings
`<p> `- paragraph
`<title>` - title of the HTML page
`<br>` - break line (no closing tag)

`<a href = "www.google.com">This is a link to Google</a>` - HTML Links
`<img src="www.youyube.com" alt= "Image from Youtube">` - Add an Image\
`<style>` - add style to HTML elements

---

**Block level Elements:**

- A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
- A block-level element always takes up the full width available (stretches out to the left and right as far as it can).
  `<table> <p> <ul> <ol> <div> <nav> <header> <footer> <form>`

`<div>`- Used as a container for other HTML elements.

**Inline Elements:**

- An inline element does not start on a new line.
- An inline element only takes up as much width as necessary.

`<a> <img> <input> <br> <span> <script> <sup> <sub>`

`<span>` - element is an inline container used to mark up a part of a text, or a part of a document.
